![Preview](https://i.ibb.co/pxh9YZK/image.png)

## Como usar
1. Clonar/Descargar el repositorio.
2. Ejecutar  ``` npm install ```.
3. Ejecutar ```npm start``` para activar el puerto del servidor de desarrollo local 3000.(http://localhost:3000).
